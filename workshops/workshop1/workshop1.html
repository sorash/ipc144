Part A - Introduction

Getting Started

Workshop 1


In this workshop, you write and execute your first C program on two different platforms.


Learning Outcomes

Upon successful completion of this workshop, you will be able to

  • use operating system utilities to edit, compile and run programs on a variety of platforms
  • transfer source code between a local computer and a remote one

Specifications

Login to a remote Linux host.  If you have not created an account on this host, ask your instructor for help. 

Open the course notes at the section on Compilers in a separate window as your reference. 

Enter the following source code using a text editor and call your program "helloWorld.c": 

 // Getting Started - Workshop 1
 // hello.c

 int main(void) {
     printf("Hello world\n"); 
 }

Compile and run this program using the Linux compiler. 

Transfer a copy of this source code from the remote computer to your local computer.

Re-compile and re-run this program on your local computer using the Windows compiler.


Prepare a Typescript

Create a typescript on the remote Linux host using the following commands: 

 + At the prompt, type: script
 + At the prompt, type: whoami
 + At the prompt, type: cat hello.c
 + At the prompt, type: gcc hello.c
 + At the prompt, type: a.out
 + At the prompt, type: exit

This will produce a file named "typescript" in your current directory.  Transfer a copy of this file to your local computer.
