 IPC144

Part D - Data Structures

Arrays

Workshop 7


In this workshop, you are to write a program that processes a list of data values.


Learning Outcome

Upon successful completion of this workshop, you will have demonstrated the abilities to

  • use one-dimensional arrays
  • validate user input
  • format program output

Grocery Store Inventory

Design and code a program that lists the packaged goods in a grocery store and calculates their total value.  Your program prompts the user for and accepts each item's

  • barcode
  • price
  • quantity in stock

Your program stops accepting data when the user enters 0 for the barcode.

Your program uses arrays, includes functions to validate the input and calculates the total value of the inventory.  You may assume that there are no more than 100 items to process, but write your code so that you can change this limit by editing a single statement.

Preface each function header with a comprehensive description of the function's purpose, the function's parameters and the function's return value.

The output from your program looks like: 

 Grocery Store Inventory
 =======================
 Barcode  :  6300090801
 Price    :  3.25
 Quantity :  5
 Barcode  :  6380613916
 Price    :  4.45
 Quantity :  15
 Barcode  :  6038367157
 Price    :  3.95
 Quantity :  10
 Barcode  :  0

             Goods in Stock
             ==============
 Barcode    Price   Quantity   Value 
 -----------------------------------
 6300090801  3.25          5   16.25
 6380613916  4.45         15   66.75
 6038367157  3.95         10   39.50
                              ------
 Total value goods in stock   122.50

Books in Stock

Design and code a program that lists the books in stock in a local bookstore and calculates their total value.  Your program prompts the user for and accepts each book's

  • ISBN
  • price
  • quantity in stock

Your program stops accepting data when the user enters 0 for the ISBN.

Your program uses arrays, includes functions to validate the input and calculates the total value of the books in stock.  You may assume that there are no more than 100 titles to process, but write your code so that you can change this limit by editing a single statement.

Preface each function header with a comprehensive description of the function's purpose, the function's parameters and the function's return value.

The output from your program looks something like:

Work in Progress (Difficult)

Design and code a program that lists a company's work in progress in project order.  Your program prompts the user for and accepts each project's

  • project number
  • hourly rate
  • hours worked

Your program stops accepting data when the user enters 0 for the project number.

Your program uses arrays, includes functions to validate the input and to sort the information by project number and calculates the total work in progress.  You may assume that there are no more than 100 projects to process, but write your code so that you can change this limit by editing a single statement.

Preface each function header with a comprehensive description of the function's purpose, the function's parameters and the function's return value.

The output from your program looks like: 

 Work in Progress

 Hours Spent :  35

 3289    120.00     35    4200.00
                         --------
 Total Work in Progress  13712.50
