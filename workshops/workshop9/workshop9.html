Part D - Data Structures

Structures

Workshop 9


In this workshop, you are to complete a program that declares a structure type and defines functions that access objects of that type. 


Learning Outcome

Upon successful completion of this workshop, you will have demonstrated the abilities to

  • design a structure type
  • access the members of an object of structure type
  • pass an object of structure type to a function

Fraction Simplifier

The following fraction simplifier program accepts a fraction, simplifies it, and then displays the result.  Your task is to complete this program.

Your solution declares a Fraction type, which holds the numerator and denominator as separate members and includes the definitions of three functions with the following prototypes

  • void enter(struct Fraction*);
  • void simplify(struct Fraction*);
  • void display(const struct Fraction*);

Complete the following program by adding the missing code:

 // Fraction Simplifier
 // w9.c

 #include <stdio.h>

 // insert Fraction declaration first

 // insert function prototypes here

 int main(void)
 {
         struct Fraction f;

         printf("Fraction Simplifier\n");
         printf("===================\n");

         enter(&f);
         simplify(&f);
         display(&f);
 }

 // insert your function definitions here

The output from your completed program looks something like

 Fraction Simplifier
 ===================
 Numerator:   4
 Denominator: 16

 1 / 4

Your enter() function receives the address of a Fraction object, accepts from the user the numerator and denominator and stores those values in the object's members. 

Your simplify() function receives the address of a Fraction object and reduces the numerator and denominator, if possible.  For a complete solution you need to design an algorithm that simplifies a fraction. 

Your display() function receives the address of a Fraction object and displays the object's numerator and denominator. 

Test several fractions to ensure that your algorithm works for all cases.  Include negative as well as zero values in your test cases.

Sales Records (Difficult)

The following sales records program accepts sales data from the user, adds the sales records, calculates the tax, if any, and displays the results.  Some items are taxable, while others are not.

Your solution declares the Sale type and defines an array of objects of that type.  A Sale type holds the quantity sold, the unit price and the taxable status.  Your solution also include two functions with the following prototypes:

  • int enter(struct Sale s[], int maxno);
  • void total(const struct Sale s[], int no_sales);

Complete the following program that calls your functions:

 // Sales Records
 // w9.c

 #include <stdio.h>
 const int MAX_SALES = 10;

 // insert your Sale declaration here

 // insert function prototypes here

 int main(void)
 {
     int noSales;
     struct Sale sale[MAX_SALES];

     printf("Sale Records \n");
     printf("=============\n");

     no_Sales = enter(sale, MAX_SALES);
     total(sale, noSales);
 }

 // insert your function definitions here

The output from your solution looks something like

 Sale Records
 =============
 Quantity :  2
 Unit Price: 1.99
 Taxable:    y
 Quantity :  1
 Unit Price: 2.99
 Taxable:    n
 Quantity :  5
 Unit Price: 1.00
 Taxable:    n
 Quantity:   0

 Subtotal    11.97
 HST (13%)    0.52
 Total       12.49

Your enter() function receives the address of an array of Sale objects and an int that holds the maximum number of elements in the array.  The function accepts the data values for each sales record and stores that data in an object of Sale type.  The function stops accepting input once the user has entered a 0 value for the quantity sold.  The function returns the number of elements stored in the array.

Your total() function receives the address of the start of an array of Sale types in a pointer parameter and the number of elements in the array in a parameter of int type.  The function totals the sales before tax, the hst and the after-tax total and displays these results.
